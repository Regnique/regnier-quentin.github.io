Dashboard Covid-19

Présentation du projet

Pour ce projet qui s'est déroulé au troisième semestre, nous devions créer un dashboard lié au Covid-19 dans la région de Meurthe-et-Moselle (54). Ce tableau de bord a été réalisé avec Chart.js, l'ensemble a été codé en PHP, JavaScript, HTML et CSS.

Dashboard

Un dashboard donne les informations les plus essentielles. Le projet nécessitait des interactions avec l'utilisateur, c'est pourquoi il peut rentrer une date dans un champ de saisie pour que la requête SQL change et mette à jour le graphique.

Chart.js

Chart.js permet de faire des graphiques très simplement. Le PHP ainsi que les requêtes SQL m'ont permis de lier directement les graphiques à la base de données.
